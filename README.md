# JXPlatforms Official Website

A futuristic, high-performance landing page for JXPlatforms, built with React, Vite, and Tailwind CSS.

## Features
- **Modern UI/UX**: Glassmorphism, 3D-style abstract shapes, and smooth animations.
- **Bilingual**: Full English and Albanian (SQ) support.
- **Responsive**: Fully optimized for Desktop, Tablet, and Mobile.
- **Tech Stack**: React 18, Vite, Tailwind CSS v3, Framer Motion.

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Deployment
To deploy to GitHub Pages, you can use the `dist` folder generated by the build command.
